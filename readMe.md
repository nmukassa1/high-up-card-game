* Game Mechanics 
- There are two players 
- Rule: First player to win 5 rounds win.
- Each player has to guess if the next drawn card is going to be higher or lower than the card facing up on the table

1. Shuffle card
2. Player 1 guesses higher or lower
3. Card if drawn and compared to guess
4. Player either wins or lose
5. Update UI
6. Switch turns
7. New round
